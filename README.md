Hello Git

Autor: Erick De Jesús Hernández García
Proyecto: Repositorio de ejemplo
Descripción: Este repositorio se creó con el propósito de practicar y dominar el uso de Git y GitHub en un entorno controlado. Incluye archivos simples destinados a comprender el flujo básico de trabajo, el versionado y la organización de proyectos.

Objetivos del proyecto

Aplicar conceptos fundamentales de control de versiones.

Practicar la creación, modificación y seguimiento de archivos.

Fortalecer el uso de Git para futuros desarrollos profesionales.

Tecnologías utilizadas

Git – Control de versiones

GitHub – Gestión y alojamiento del repositorio

Python – Scripts de ejemplo
